![](https://img.shields.io/badge/React_·_Flask_·_MySQL_·_JWT-0d1117?style=flat-square&logo=react&logoColor=61DAFB)
![](https://img.shields.io/badge/Status-Complete-00C853?style=flat-square)

**💀 Problem:** Hospitals running on paper records and spreadsheets — no access control, no digital prescriptions, zero audit trail.

**✅ Solved:** Role-isolated dashboards for Doctor · Patient · Admin. JWT + RBAC so each role sees *only* their data. Appointments, prescriptions, medical records, billing — all in one platform.

**🔑 Hard part:** Enforcing data isolation at the API level — not just hiding UI buttons.
